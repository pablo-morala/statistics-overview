# Statistics overview
Hello! This repository contains a Jupyter Notebook focused on basic statistical concepts implemented in Python. The notebook covers fundamental topics such as descriptive statistics, hypothesis testing, statistical inference, linear regression, and p-value calculations.

Its main goal is to help users grasp essential statistical ideas and understand how to perform the related calculations using Python code. This practical approach combines theory and computation, making it easier to learn and apply statistics through hands-on examples.

By working with this notebook, you will see how Python can be used to explore data, carry out statistical tests, build simple models, and interpret results, providing a solid foundation in basic statistics.
